#### 框架

> React/Vue 中列表组件中为什么要写 key，其作用是什么？

    key 是给每一个虚拟节点的唯一 id，依靠 key 可以更准确、更快的拿到旧虚拟节点对应的虚拟节点。
    如果不使用 key， 会使用最大限度减少动态元素并且尽可能的尝试就地修改/复用相同类型元素的算法。
    而使用 key时，它会基于 key 的变化重新排列元素顺序，并且会移除 key 不存在的元素。
